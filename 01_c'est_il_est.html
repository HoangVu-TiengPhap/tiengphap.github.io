Exercice : C'est / Il est (Elle est...)

Complétez les phrases sau bằng cách sử dụng C'est, Ce sont, Il est, Elle est, Ils sont, hoặc Elles sont.
(Điền vào chỗ trống C'est, Ce sont, Il est, Elle est, Ils sont, hoặc Elles sont.)

Ví dụ:
... est un musicien. un musicien.
... est excellent. excellent.
(Đây là một nhạc sĩ. Anh ấy thì tuyệt vời.)

Gợi ý:
  • Sử dụng C'est / Ce sont + mạo từ (un, une, des) + danh từ.
  • Sử dụng Il est / Elle est / Ils sont / Elles sont + tính từ (malade, intelligent, célèbre...) hoặc danh từ chỉ nghề nghiệp (journaliste, médecin...) khi không có mạo từ đi kèm.
1. est un journaliste. est italien.
(... là một nhà báo. ... là người Ý.)
2. est une hôtesse. est jolie.
(... là một tiếp viên. ... thì xinh đẹp.)
3. est un médecin. est connu.
(... là một bác sĩ. ... thì nổi tiếng.)
4. est un étudiant. est intelligent.
(... là một sinh viên. ... thì thông minh.)
5. est un écrivain.
est célèbre.
(... là một nhà văn. ... thì nổi tiếng.)
6. est une avocate.
est américaine.
(... là một luật sư nữ. ... là người Mỹ.)
